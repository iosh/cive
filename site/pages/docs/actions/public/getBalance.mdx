# getBalance

Get the balance of the given account, identified by its address.

## Usage

:::code-group

```ts twoslash [example.ts]
import { publicClient } from "./client";
// [!code focus:99]
const balance = await publicClient.getBalance({
  address: "cfx:...",
});
```

```ts twoslash [client.ts] filename="client.ts"
// [!include ~/snippets/publicClient.ts]
```

:::

## Returns

`bigint`

The balance of the address in drip.

## Parameters

### address

- **address** `Address`

The address to check for balance

```ts twoslash
// [!include ~/snippets/publicClient.ts]
// ---cut---
const balance = await publicClient.getBalance({
  address: "cfx:...", // [!code focus]
});
```

### epochTag (optional)

- **epochTag:** `latest_state`| `latest_confirmed`| `latest_checkpoint` | `earliest`
- **Defaut:** `latest_state`

```ts twoslash
// [!include ~/snippets/publicClient.ts]
// ---cut---
const balance = await publicClient.getBalance({
  address: "cfx:...",
  epochTag: "latest_state", // [!code focus]
});
```

### epochNumber (optional)

- **Type:** `bigint`

```ts twoslash
// [!include ~/snippets/publicClient.ts]
// ---cut---
const balance = await publicClient.getBalance({
  address: "cfx:...",
  epochNumber: 1n, // [!code focus]
});
```

## Tips

You can convert the balance to ether units with `formatCFX`.

```ts twoslash
// [!include ~/snippets/publicClient.ts]
// ---cut---
import { formatCFX } from "cive";
const balance = await publicClient.getBalance({
  address: "cfx:...",
});
const balanceAsCFX = formatCFX(balance); // [!code focus:2]
     // "99"
```
