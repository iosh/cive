# getAccountPendingTransactions

Get ht pending transactions in pool by address.

## Usage

:::code-group

```ts twoslash [example.ts]
import { publicClient } from "./client";
// [!code focus:99]
const pendingTx = await publicClient.getAccountPendingTransactions({
  address: "cfx:...",
});
```

```ts [client.ts] filename="client.ts"
// [!include ~/snippets/publicClient.ts]
```

:::
