# getCollateralForStorage

Get the size of the collateral storage of a given address, in bytes.

:::code-group

```ts twoslash [example.ts]
import { publicClient } from "./client";
// [!code focus:99]
const balance = await publicClient.getCollateralForStorage({
  address: "cfx:...",
});
```

```ts twoslash [client.ts] filename="client.ts"
// [!include ~/snippets/publicClient.ts]
```

:::

## Returns

`bigint`

The size of the collateral storage in bytes.

## Parameters

### address

- **Type:** `Address`

The address to check for collateral storage

```ts twoslash
// [!include ~/snippets/publicClient.ts]
// ---cut---
const balance = await publicClient.getCollateralForStorage({
  address: "cfx:...", // [!code focus]
});
```

### epochTag (optional)

- **Type:** `latest_state`| `latest_confirmed`| `latest_checkpoint` | `earliest`
- **Defaut:** `latest_state`

```ts twoslash
// [!include ~/snippets/publicClient.ts]
// ---cut---
const balance = await publicClient.getCollateralForStorage({
  address: "cfx:...",
  epochTag: "latest_state", // [!code focus]
});
```

### epochNumber (optional)

- **Type:** `bigint`
