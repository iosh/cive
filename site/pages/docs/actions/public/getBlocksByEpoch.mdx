# getBlocksByEpoch

Get the block hashes in the specified epoch.

## Usage

:::code-group

```ts [example.ts]
import { publicClient } from "./client";
// [!code focus:99]
const hashes = await publicClient.getBlocksByEpoch();
```

```ts [client.ts] filename="client.ts"
// [!include ~/snippets/publicClient.ts]
```

:::

## Parameters

### epochTag (optional)

- **Type** `latest_mined`|`latest_state`|`latest_confirmed`|`latest_checkpoint` | `earliest`
- **Defaut** `latest_state`

```ts
const hashes = await publicClient.getBlocksByEpoch({
  epochTag: "latest_state", // [!code focus]
});
```

### epochNumber (optional)

- **Type** `bigint`

```ts
const hashes = await publicClient.getBlocksByEpoch({
  epochNumber: 1n, // [!code focus]
});
```
