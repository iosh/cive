# getStakingBalance

Get the stacking balance of the accout, identified by its address.

## Usage

:::code-group

```ts [example.ts]
import { publicClient } from "./client";
// [!code focus:99]
const balance = await publicClient.getStakingBalance({
  address: "cfx:...",
});
```

```ts [client.ts] filename="client.ts"
// [!include ~/snippets/publicClient.ts]
```

:::

## Returns

`bigint`

The staking balance in drip.

## Parameters

### address

- **address:** `Address`

```ts
const balance = await publicClient.getStakingBalance({
  address: "cfx:...", // [!code focus]
});
```

### epochTag (optional)

- **Type:** `latest_state`| `latest_confirmed`| `latest_checkpoint` | `earliest`
- **Defaut:** `latest_state`

```ts
const balance = await publicClient.getStakingBalance({
  address: "cfx:...",
  epochTag: "latest_state", // [!code focus]
});
```

### epochNumber (optional)

- **Type:** `bigint`

```ts
const balance = await publicClient.getStakingBalance({
  address: "cfx:...",
  epochNumber: 1n, // [!code focus]
});
```
