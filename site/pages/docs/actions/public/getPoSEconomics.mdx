# getPoSEconomics

Get the PoS economics summary info.

## Usage

:::code-group

```ts twoslash [example.ts]
import { publicClient } from "./client";
// [!code focus:99]
const PoSEconomics = await publicClient.getPoSEconomics();
```

```ts [client.ts] filename="client.ts"
// [!include ~/snippets/publicClient.ts]
```

:::

## Parameters

### epochTag (optional)

- **Type:** `latest_state`| `latest_confirmed`| `latest_checkpoint` | `earliest`
- **Defaut:** `latest_state`

```ts
const PoSEconomics = await publicClient.getPoSEconomics({
  epochTag: "latest_state", // [!code focus]
});
```

### epochNumber (optional)

- **Type:** `bigint`

```ts
const PoSEconomics = await publicClient.getPoSEconomics({
  epochNumber: 1n, // [!code focus]
});
```
