# getAddresses

Returns a list of account addresses owned by the wallet or client.

## Usage

:::code-group

```ts twoslash [example.ts]
import { walletClient } from "./client";

const accounts = await walletClient.getAddresses(); // [!code focus:99]
// ['cfx:...', 'cfx:...']
```
