# getAdmin

Get the admin of the specified contract

## Usage

:::code-group

```ts  [example.ts]
import { publicClient } from "./client";
// [!code focus:99]
const contractAddress = await publicClient.getAdmin({
  address: "cfx:...",
});
```

```ts  [client.ts] filename="client.ts"
// [!include ~/snippets/publicClient.ts]
```

:::

## Returns

`Address` | `null`

Return the admin of the specified contract address or `null` if the contract does not exist.

## Parameters

### address

- **Type:** `Address`

The address to check for collateral storage

```ts 
const contractAddress = await publicClient.getAdmin({
  address: "cfx:...", // [!code focus]
});
```

### epochTag (optional)

- **Type:** `latest_state`| `latest_confirmed`| `latest_checkpoint` | `earliest`
- **Defaut:** `latest_state`

```ts 
const contractAddress = await publicClient.getAdmin({
  address: "cfx:...",
  epochTag: "latest_state", // [!code focus]
});
```

### epochNumber (optional)

- **Type:** `bigint`

```ts 
const contractAddress = await publicClient.getAdmin({
  address: "cfx:...",
  epochNumber: 1n, // [!code focus]
});
```
