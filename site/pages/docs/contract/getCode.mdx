# getCode

Get the code of the specified contract. If contract not exist will return `undefined`.

## Usage

:::code-group

```ts  [example.ts]
import { publicClient } from "./client";
// [!code focus:99]
const bytecode = await publicClient.getCode({
  address: "cfx:...",
});
```

```ts  [client.ts] filename="client.ts"
// [!include ~/snippets/publicClient.ts]
```

:::

## Returns

Return `Hex` if the address has code; otherwise, return `undefined`.

## Parameters

### address

- **Type:** `Address`

The address to check for collateral storage

```ts 
const contractAddress = await publicClient.getCode({
  address: "cfx:...", // [!code focus]
});
```

### epochTag (optional)

- **Type:** `latest_state`| `latest_confirmed`| `latest_checkpoint` | `earliest`
- **Defaut:** `latest_state`

```ts 
const contractAddress = await publicClient.getCode({
  address: "cfx:...",
  epochTag: "latest_state", // [!code focus]
});
```

### epochNumber (optional)

- **Type:** `bigint`

```ts 
const contractAddress = await publicClient.getCode({
  address: "cfx:...",
  epochNumber: 1n, // [!code focus]
});
```
