## Using Canary Version 

Cive is currently under active development, If you want to try it, You can install the `canary` version.

:::code-group

```bash [pnpm]
pnpm add cive@canary
```

```bash [npm]
npm install cive@canary
```

```bash [yarn]
yarn add cive@canary
```

```bash [bun]
bun add cive@canary
```
:::
