# Public Client

A Public Client is interface to the [Conflux Core Space JSON-RPC](https://doc.confluxnetwork.org/docs/core/build/json-rpc/). See the all supported [Public Actions](/docs/actions/public/introduction) for more details.

## Import

```ts twoslash
import { createPublicClient } from "cive";
```

## Usage

```ts twoslash
// [!include ~/snippets/publicClient.ts:imports]
const publicClient = createPublicClient({
  chain: mainnet,
  transport: http(),
});
```

## Parameters

### transport

- **Type** [Transport](#/docs/clients/transports/http)

The [Transport](/docs/clients/transports/http) of the Public Client.

```ts
const publicClient = createPublicClient({
  chain: mainnet,
  transport: http(), // [!code focus]
});
```

## cacheTime(optional)

- **Type** `number`
- **Default** `4000`

Time (in ms) that cached data will remain memory.

```ts
const publicClient = createPublicClient({
  cacheTime: 10_000, // [!code focus]
  chain: mainnet,
  transport: http(),
});
```

### key(optional)

- **Type** `string`
- **Default** `public`

The key of the Public Client.

```ts
const publicClient = createPublicClient({
  key: "public", // [!code focus]
  chain: mainnet,
  transport: http(),
});
```

### name(optional)

- **Type** `string`
- **Default** `Public Client`

The name of the Public Client.

```ts
const publicClient = createPublicClient({
  name: "Public Client", // [!code focus]
  chain: mainnet,
  transport: http(),
});
```

### pollingInterval(optional)

- **Type** `number`
- **Default** `4000`

```ts
const publicClient = createPublicClient({
  pollingInterval: 4_000, // [!code focus]
  chain: mainnet,
  transport: http(),
});
```
